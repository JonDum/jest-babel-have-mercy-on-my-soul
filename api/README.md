

## Data


data/faa/*.txt are directly from FAA

https://www.faa.gov/licenses_certificates/aircraft_certification/aircraft_registry/releasable_aircraft_download/


data/airports.tsv is from https://openflights.org/data.html converted to tsv via OpenRefine

It also has several missing airports/timezones THAT ARE NOT IN THE ORIGINAL (do not replace directly).


data/airplanes.csv is from http://ourairports.com/data/
